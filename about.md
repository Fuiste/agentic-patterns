**Agentic Patterns** is a collection of development patterns for working effectively with AI coding agents — tools like Cursor, Claude Code, Codex, and others that write code alongside you.

These aren't theoretical frameworks. They're patterns that emerged from building real software with agentic tools, refined through repetition until they reliably produce good results.

## What you'll find here

Each post documents a specific pattern: the problem it solves, when to use it, how to implement it, and the failure modes to watch for. The goal is to give you something you can apply to your own work immediately.

## Who writes this

I'm [William Pelrine](https://github.com/Fuiste). I build software with AI coding agents daily and started writing these patterns down because I kept explaining the same things to different people.

If you want to discuss a pattern or share one of your own, open an issue on the [GitHub repo](https://github.com/Fuiste/agentic-patterns).
